# issue_002: Flask-Session のバックエンド

## 決定（クローズ）

- バックエンドは filesystem（セッションIDごとに別ファイル）を使用。複数プロセスでも各セッションファイルは独立しているため問題なし。
- セッションディレクトリは環境変数 `ACCMAN_SESSION_DIR` で設定する。
  - 未設定時のデフォルト: `<プロジェクトルート>/var/sessions/`（開発用）
  - 本番環境では Flask 起動ユーザのホームディレクトリ下（`/var/lib/<user>/sessions` 等）を設定すること。
- `/tmp` は使用しない（OS 再起動でクリアされるため不適切）。
